# Machine Learning Projects (Regression, Classification & Optimization)

## Overview
This repository contains implementations of fundamental machine learning models and optimization techniques developed during university coursework.

The focus lies on understanding core ML concepts, implementing models, and evaluating their performance on real datasets.

## Contents

### HW1 – Regression & Optimization
- Linear Regression (univariate & multivariate)
- Polynomial Regression
- Logistic Regression
- Gradient Descent (Rastrigin optimization)
- Model evaluation (MSE, accuracy, cross-entropy)

### HW2 – Model Comparison
- Neural Networks
- Random Forests
- Support Vector Machines (SVM)
- Model comparison and performance analysis
- Hyperparameter tuning and generalization analysis

## Key Features
- Custom design matrices and feature transformations
- Model evaluation using multiple metrics (MSE, accuracy, loss)
- Analysis of overfitting and model complexity
- Implementation of optimization algorithms (gradient descent)

## Tech Stack
- Python
- NumPy
- scikit-learn
- Matplotlib

## Highlights
- Achieved up to 100% classification accuracy on selected datasets
- Compared multiple ML models and analyzed trade-offs between them
- Implemented regression models and optimization methods from scratch

# HW1 – Regression & Optimization

## Overview
This project focuses on implementing and analyzing fundamental machine learning models including regression techniques and optimization algorithms.

## Topics Covered

### Linear Regression
- Univariate and multivariate regression
- Analytical solution using normal equations
- Pearson correlation analysis
- Model evaluation using Mean Squared Error (MSE)

### Polynomial Regression
- Feature expansion using polynomial design matrices
- Analysis of model complexity and overfitting

### Logistic Regression
- Classification using scikit-learn
- Custom feature engineering for decision boundaries
- Evaluation using accuracy and cross-entropy loss
- Regularization (L1 / L2)

### Gradient Descent
- Implementation of gradient descent with learning rate decay
- Optimization of the Rastrigin function
- Comparison between analytical and numerical gradients

## Key Learnings
- Understanding regression models mathematically and practically
- Designing feature transformations
- Evaluating model performance using multiple metrics
- Handling optimization challenges in non-convex functions

## Results
- Strong linear relationships identified using correlation analysis
- Improved regression performance with polynomial features
- High classification accuracy through feature engineering and tuning
- Successful optimization of complex functions using gradient descent

## Tech Stack
- Python
- NumPy
- scikit-learn
- Matplotlib
